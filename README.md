

# 🚗 Car Rental App | تطبيق تأجير السيارات


> 🛠️ **المشروع قيد التطوير حالياً**

تطبيق ويب احترافي لتأجير السيارات، يتيح للمستخدمين اختيار مدة الإيجار (يوم، أسبوع، شهر، أو سنة) مع إمكانية إضافة خدمات إضافية مثل الإنترنت داخل السيارة، مقعد للأطفال، وغيرها من الميزات المخصصة لتجربة مريحة ومتكاملة.

## 💡 المميزات

- اختيار مدة الإيجار حسب الحاجة: يوم / أسبوع / شهر / سنة.
- إضافة خيارات إضافية مثل:
  - الإنترنت داخل السيارة.
  - مقعد للأطفال.
  - ميزات أخرى مخصصة.
- واجهة مستخدم أنيقة وسهلة الاستخدام.
- دعم كامل للتجاوب مع جميع الشاشات.

## 🛠️ التقنيات المستخدمة

- **Next.js** – لبناء تطبيق الويب بتقنيات React وميزات SSR/SSG.
- **Tailwind CSS** – لتنسيق الواجهة بشكل مرن وسريع.
- **shadcn/ui** – لمكونات واجهة مستخدم جاهزة وعصرية تعتمد على Radix UI.

## 🚀 الهدف من المشروع

تقديم منصة رقمية سهلة وسريعة لتأجير السيارات مع خيارات مخصصة تناسب جميع الاحتياجات، سواء كانت رحلات قصيرة أو طويلة.

# 🚗 Car Rental App

> 🛠️ **This project is currently under development**

A modern web app for renting cars with flexible durations (daily, weekly, monthly, yearly) and customizable add-ons like in-car internet, child seat, and more.

## 💡 Features

- Choose rental duration: Day / Week / Month / Year.
- Add optional services like:
  - In-car Wi-Fi.
  - Child seat.
  - Other convenience options.
- Clean and user-friendly interface.
- Fully responsive design for all devices.

## 🛠️ Tech Stack

- **Next.js** – Powerful React framework with SSR/SSG.
- **Tailwind CSS** – Utility-first styling framework.
- **shadcn/ui** – Modern UI components built on top of Radix UI.

## 🚀 Project Goal

To offer a flexible, fast, and modern platform for renting vehicles tailored to different customer needs, from quick trips to long-term use.
